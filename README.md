## Introduction
That is my task for recruitment process. I did all the steps (1. Display result table of Ekstraklasa, 2. Display more information, 3. Add season filter to results table, 4. Add subpage with more info about chosen match, 5. Add some features, improve styling, refactoring)


## Technologies Used
* React
* React Bootstrap
* React Hooks
* Axios for fetch API
* Cors anywhere heroku app (https://cors-anywhere.herokuapp.com)
